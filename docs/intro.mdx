# Learn Ameide

Three destinations:

- **[Documentation](/docs/documentation/overview)** — product
  documentation for using Ameide inside a customer organization.
  Strategy, Architecture, Delivery, Change, Support.
- **[Training & Certifications](/docs/training/overview)** — learning
  paths, courses, certifications, and playbooks.
- **[Partners](/docs/partners/overview)** — functional documentation
  for the Partners module: Consulting Partners and Technology Partners.

If you're new, start with [Getting Started](/docs/getting-started/what-is-ameide).
Cross-cutting platform topics live under
[Administration](/docs/administration/identity-and-sso).

- **Platform operations and GitOps** —
  [docs.ameide.io](https://docs.ameide.io)
- **The product itself** —
