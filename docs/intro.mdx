# Learn Ameide

This site is for people using Ameide day to day — tenant admins, product
users, and anyone onboarding a team onto the platform.

## What's here

- **Getting started** — what Ameide is, how to invite your team, and the
  workspace concepts every user should know.
- **Products** — per-product walkthroughs for Identity, Support, Sales,
  Marketing, and Threads.
- **Administration** — billing, roles and permissions, audit log.
- **Tutorials** — end-to-end playbooks.

## What's _not_ here

API reference, protobuf schemas, and integration guides live on the
[Developer portal](https://developers.ameide.io).
