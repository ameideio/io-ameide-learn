# Partners

Functional documentation for the Ameide **Partners** module — the
in-platform tooling partners use to run their practice on Ameide.

Two partner types ship in the module:

- **[Consulting Partners](/docs/partners/consulting/overview)** — firms
  that deliver implementation, advisory, and managed services on Ameide.
- **[Technology Partners](/docs/partners/technology/overview)** — ISVs
  and integrators that extend Ameide with their own technology, listings,
  and certifications.

For the partner *program* itself (tiers, deal registration, becoming a
partner), see [ameide.io/partners](https://ameide.io/partners).
