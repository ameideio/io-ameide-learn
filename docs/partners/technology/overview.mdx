# Technology Partners

For ISVs and integrators extending Ameide with their own technology,
listings, and certifications.

## What's covered

- **Build** — Integration patterns, SDKs, webhooks.
- **List** — Marketplace listing, branding, and review.
- **Certify** — Technical certification for integrations.
- **Co-sell** — Joint go-to-market motions.

For the program meta (tiers, application, agreements), see
[ameide.io/partners](https://ameide.io/partners). For API and protobuf
reference, see [developers.ameide.io](https://developers.ameide.io).
