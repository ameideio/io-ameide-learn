# What is Ameide

Ameide is a unified workspace for customer-facing teams. A single tenant
runs Identity, Support, Sales, Marketing, and Threads on top of one
shared contract — the same protobuf schemas that power the
[Developer portal](https://ameideio.github.io/io-ameide-developers/).

:::info
This page is a placeholder. Content will land as product walkthroughs
get drafted.
:::

## Key concepts

- **Tenant** — the top-level boundary; your organisation.
- **User** — a person in a tenant, with one or more roles.
- **Workspace** — the product-level surface a user works in day to day.
